Install Cursive PD Tracker

The extension runs on your Chrome browser, fetches public data from marketplaces for products you choose to monitor, and updates your dashboard automatically.

1

Make sure you have Google Chrome

The extension only works in Chrome on Windows / Mac / Linux. If you don't have it, download Chrome here.

2

Download the extension

Click below to download the installer (~3 MB). Run it once — it will set up Chrome automatically.

⬇ Download for Windows

(Coming soon: Chrome Web Store one-click install for all platforms)

3

Sign in once with your account

After install, click the Cursive icon in your Chrome toolbar. Sign in with the same email + password you use here.

4

That's it — leave Chrome running

The extension runs quietly in the background. It updates your data automatically every 6 hours. You'll see the 🟢 Tracker online badge on your dashboard.

If your PC is off, no data updates — just turn it back on and the extension picks up where it left off.
