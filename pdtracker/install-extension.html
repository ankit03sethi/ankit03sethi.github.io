Install Cursive PD Tracker

Pick the install method that works for you. The extension runs on your Chrome and updates your dashboard automatically.

RECOMMENDED · Windows

Option 1 — Auto-installer

Downloads + extracts + opens Chrome for you. ~30 seconds.
⬇ Download Auto-Installer (.bat)
  1. Click the download button above — saves a small .bat file.
  2. Double-click the downloaded file. (If Windows warns: click "More info" → "Run anyway".)
  3. It downloads the extension, extracts it, and opens Chrome's Extensions page.
  4. In Chrome: toggle Developer mode ON (top-right) → click Load unpacked → paste the path (already copied to clipboard) → Enter.
  5. Click the Cursive icon in Chrome toolbar → sign in with your email/password.

Option 2 — Manual sideload

For Mac, Linux, or if Option 1 doesn't work.
⬇ Download Extension (.zip)
  1. Download the ZIP and unzip it to any folder you remember (e.g. Desktop).
  2. Open Chrome → go to chrome://extensions in the address bar.
  3. Turn ON Developer mode (top-right toggle).
  4. Click Load unpacked → select the unzipped folder.
  5. Click the Cursive icon in toolbar → sign in.

Option 3 — Chrome Web Store COMING SOON

True 1-click install + silent auto-updates. Submitted for Google approval (1-3 working days).

Once approved, this will be the easiest way. Until then, use Option 1.
