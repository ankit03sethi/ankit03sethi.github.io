✎ Update existing product data

Edit Seller Code and Seller Name for products you've already added. Platform, ID Name, Product ID, and Note columns stay locked — if you want to track a new product, use + Add new products instead.

Step 1: Download your current data

We'll generate the same Excel format as the new-products template, pre-filled with your existing rows. Columns A (Platform), B (ID Name), C (Product ID), D (Note) are locked. Edit E (Seller Code) and F (Seller Name) as needed, then save.

Step 2: Upload your edited file

We'll match each row by Platform + Product ID and update only the Seller Code and Seller Name.
