How it works

1
Install & sign in
Double-click the installer. Silent install (~10 sec). Sign in with your cursive.world email. App runs in system tray, invisible otherwise.
2
App auto-creates a folder per company
The app reads your company list from cursive.world and creates C:\Users\<you>\SellerOps\<CompanyName>\ for each one.
3
Point each Chrome profile to its company folder
One-time: in each Chrome profile Settings → Downloads → set Location to the matching SellerOps\<Company>\ folder. Takes 30 seconds per profile.
4
Just download files — done
Click Download on Amazon / Flipkart / Meesho / FirstCry / Myntra seller panels. Files auto-flow to Cursive Analyst, get tagged with company + platform, appear in your SKU Wise dashboard within seconds.

✔ Zero maintenance

• Runs invisibly in system tray, auto-starts with Windows.
• When we ship an update, it auto-installs silently.
• Never touches your Chrome logins or marketplace accounts.
• Adds a new company later? Add on cursive.world; folder appears automatically.
